# MindFlow - AI Therapy Assistant Web Application

## Complete Project Package

This package contains all documentation and implementation files for the MindFlow AI Therapy Assistant.

### Source Code (`/src`)
- Complete React + TypeScript implementation
- All components, services, stores, and utilities
- Ready to run with `npm install && npm run dev`

### Configuration
- `package.json` - Dependencies and scripts
- `vite.config.ts` - Vite build configuration

- `tailwind.config.js` - Tailwind CSS configuration

## Quick Start

1. Extract this package
2. Navigate to project directory: `cd mindflow-ai-therapy`
3. Install dependencies: `npm install`
4. Set up Anthropic API key in `.env` file
5. Run development server: `npm run dev`
6. Open browser to `http://localhost:5173`

## Technology Stack

- **Frontend:** React 18 + TypeScript
- **State Management:** Zustand
- **Styling:** Tailwind CSS + shadcn/ui
- **Build Tool:** Vite
- **AI Integration:** Anthropic Claude API
- **Storage:** Browser localStorage
- **Export:** jsPDF + file-saver

## Key Features

- ✅ Flexible intake approaches (Crisis/Brief/Structured)
- ✅ Comprehensive clinical file tracking
- ✅ Validated screening tools (PHQ-9, GAD-7, ASRS, C-SSRS)
- ✅ Real-time AI therapy chat
- ✅ Session management and history
- ✅ Progress tracking with charts
- ✅ Export to PDF/Markdown
- ✅ Safety protocols and crisis management
- ✅ 100% client-side data storage (privacy-first)
