# Code of Conduct

We aim to keep this project welcoming, inclusive, and professional.

## Expected behavior
- Be respectful and constructive.
- Assume good intent and provide clear, actionable feedback.
- Keep discussions focused on the work.

## Unacceptable behavior
- Harassment, discrimination, or personal attacks.
- Threats, doxxing, or sharing private information.
- Disruptive or hostile behavior in issues or PRs.

## Reporting
If you experience or witness a violation, contact the repository owner via
GitHub direct message. If you are not comfortable doing that, open an issue
and label it `conduct` with only the minimum details needed to start a private
conversation.

## Enforcement
Project maintainers may remove content or restrict participation to protect
contributors and the community.
